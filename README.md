Office x86 Installer
======================
这是一个apk+数据包的自动安装器，与微软的office没有太大关系。

起因是，office的安卓平板专用版本很多设备无法访问Google Play下载，而它安装起来又是由apk+.obb文件构成，与普通安卓程序安装不同，很多用户并不会，因此写了一个自动安装器。

<h2>适用平台</h2>
Android 4.4+
